## <a name="dev"></a>Локальный запуск проекта

```bash
# установка зависимостей
$ npm ci

# запуск сервера разработки
# с горячей перезагрузкой на localhost:3000
$ npm run dev

# генерация статичного сайта
$ npm run export
```
